Name

applyMatrix
Examples 
P3D <- "processing.opengl.PGraphics3D"
PI <- pi

settings <- function() {
    size(100, 100, P3D)
}

draw <- function() {
    noFill()
    translate(50, 50, 0)
    rotateY(PI/6)
    stroke(153)
    box(35)
    # Set rotation angles
    ct = cos(PI/9)
    st = sin(PI/9)
    # Matrix for rotation around the Y axis
    applyMatrix(ct, 0, st, 0, 0, 1, 0, 0, -st, 0, ct, 0, 0, 0, 0, 1)
    stroke(255)
    box(50)
}
Description Multiplies the current matrix by the one specified through the parameters. This is very slow because it will try to calculate the inverse of the transform, so avoid it whenever possible. The equivalent function in OpenGL is glMultMatrix().
