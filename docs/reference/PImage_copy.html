Name

PImage_copy
Description Copies a region of pixels from one image into another. If the source and destination regions aren't the same size, it will automatically resize source pixels to fit the specified target region. No alpha information is used in the process, however if the source image has an alpha channel set, it will be copied as well.

As of release 0149, this function ignores imageMode().
Syntax
pimg.copy(sx, sy, sw, sh, dx, dy, dw, dh)
pimg.copy(src, sx, sy, sw, sh, dx, dy, dw, dh)
Parameters
pimgPImage: any object of type PImage
sxint: X coordinate of the source's upper left corner
syint: Y coordinate of the source's upper left corner
swint: source image width
shint: source image height
dxint: X coordinate of the destination's upper left corner
dyint: Y coordinate of the destination's upper left corner
dwint: destination image width
dhint: destination image height
srcPImage: an image variable referring to the source image.
